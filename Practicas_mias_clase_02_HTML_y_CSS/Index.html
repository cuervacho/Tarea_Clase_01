Una pequeña autobiografía y espectativas del curso

Mi nombre es Darío Muñoz, soy de Santa Rosa provincia de La Pampa.

Mi trabajo es de oficina. Soy estudiante de Ciencias Económicas pero me apasiona la informática. Me gusta estudiar y automatizar procesos para hacer el trabajo menos pesadoy tedioso para poder así aprovechar mejor el tiempo, por ende, todo esto es para mi un híbrido entre trabajo, hobbie y además una manera de ganar unos minutos más al día para poder hacer otras cosas que me gustan y compartir más tiempo con mi familia, en resumen, robarle unos minutitos diarios a las obligaciones para dedicarlos ser feliz =).

Esta es mi primer interacción con lo que se dice Front End, me dio curiosidad y desde que decidí aprender programación sentí la necesidad de adquirir conocimientos en estas tecnologías que nos ofrece este curso de BA Multiplica x2 y la UTN. Así que espero cumplir mis espectativas y las de los profes que imparten este valioso curso.

Y desde ya,doy gracias por esta oportunidad y al profesor que tenemos por brindarnos su valioso tiempo.

LISTAS DE DEFINICIÓN

    Breve detalle de cada uno

Html :
HyperText Markup Language (Lenguaje de Marcas de Hipertexto). Es el componente más básico de la Web. Define el significado y la estructura del contenido web.
CCS :
Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje de estilos utilizado para describir la presentación de documentos HTML, es decir la apariencia de una página web.
JavaScript(JS) :
Es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase. Si bien es más conocido como un lenguaje de scripting (secuencias de comandos) para páginas web, y es usado en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat JavaScript es un lenguaje de programación basada en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa (por ejemplo programación funcional).
